# KSL

<ol>
Pull request
  <li>master 에서 토픽 브랜치를 만든다.</li>
  <li>뭔가 수정해서 커밋한다.</li>
<li>자신의 GitHub 프로젝트에 브랜치를 Push 한다.</li>
<li>GitHub에 Pull Request를 연다.</li>
<li>토론하면서 그에 따라 계속 커밋한다.</li>
<li>프로젝트 소유자는 Pull Request를 Merge 하고 닫는다.</li>
</ol>
